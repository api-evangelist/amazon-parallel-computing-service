This post announces a new observability solution for AWS Parallel Computing Service (AWS PCS) that combines Amazon Managed Grafana with Amazon Managed Service for Prometheus. The solution provides real-time monitoring dashboards offering visibility into HPC cluster performance, resource allocation, job metrics, GPU usage, and network performance. The implementation helps administrators quickly identify and resolve performance issues while optimizing costs and resource utilization.
